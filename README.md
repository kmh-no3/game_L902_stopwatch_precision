# 🎯 Precision Stopwatch Drill

ターゲット時間にぴたりと合わせるストップウォッチチャレンジゲーム。1/1000秒単位まで表示されるタイマーで、集中力と時間感覚を鍛えます。

## ✨ 特徴

- **精密タイミング計測**: ミリ秒単位（0.000秒）での計測が可能
- **集中モード**: 時間表示を隠して、感覚だけでタイミングを掴む練習
- **ターゲット調整**: ランダム生成または手動で2〜5秒の範囲で設定可能
- **成績評価**: S/A/B/Cのグレードで判定
- **履歴表示**: 直近5回の挑戦結果を記録
- **達成演出**: 高評価時の視覚的フィードバック

## 🎮 遊び方

1. **スタート**ボタンを押して計測開始
2. ターゲット時間を頭の中でイメージ
3. 狙ったタイミングで**ストップ**を押す
4. ターゲットとの差が小さいほど高評価

### 操作方法

- **スタート/ストップ**: 計測の開始と停止
- **ランダムターゲット**: 新しいターゲット時間を自動生成
- **ターゲット微調整**: スライダーで手動調整（2.00〜5.00秒）
- **集中モード**: 時間表示を隠して感覚のみで挑戦

## 📊 評価基準

- **Sランク**: 誤差0.05秒未満
- **Aランク**: 誤差0.12秒未満
- **Bランク**: 誤差0.25秒未満
- **Cランク**: 誤差0.25秒以上
